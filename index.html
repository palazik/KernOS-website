// what is kernos

Built differently.
Performs differently.

KernOS is a minimal, performance-focused Linux distribution based on CachyOS & Arch Linux. It strips away everything unnecessary and ships only what you actually need — giving you a fast, responsive system from the first boot.

Built by a kernel developer, for people who care about what runs under the hood. No desktop environment preinstalled. No unnecessary daemons. Just your hardware, the kernel, and you.

// linux-kernos

The Kernel.
Tuned to the max.

⚡

BORE Scheduler

Burst-Oriented Response Enhancer — prioritizes interactive tasks for snappy desktop and gaming performance.

sched/BORE
🔗

ThinLTO

Link-Time Optimization with Clang — whole-program optimization with minimal memory overhead. Faster binaries, same safety.

LTO_CLANG_THIN
🎯

O3 Optimizations

Maximum compiler optimization level. Every CPU cycle squeezed for raw performance across the entire kernel.

CC_OPTIMIZE_FOR_PERFORMANCE_O3
📡

BBR TCP

Google's BBR congestion control replaces Cubic. Better throughput and lower latency on modern networks.

TCP_CONG_BBR
🧩

Full Preemption

Full preemptive kernel with 1000Hz tick rate and tickless idle — ultra-low latency for real-time workloads.

PREEMPT + HZ_1000
🔲

Huge Pages Always

Transparent hugepages always enabled — reduces TLB pressure and improves memory throughput for large applications.

TRANSPARENT_HUGEPAGE_ALWAYS
📦

ZSTD Compression

Fast, modern compression for the kernel image and modules. Fast decompression means faster boot times.

KERNEL_ZSTD
🔌

sched-ext Support

eBPF-based scheduler extension framework. Load custom schedulers at runtime without recompiling the kernel.

SCHED_CLASS_EXT
🚫

No Debug Bloat

Debug symbols, KPROBES, FTRACE and SLUB debug are all stripped. Smaller image, faster boot, less overhead.

-DEBUG_INFO -FTRACE

// install

Get KernOS
running.

1

Download the ISO

Grab the latest ISO from GitHub Releases (built automatically on every commit).

bash
wget https://github.com/palazik/KernOS/releases/latest/download/kernos-*.iso
2

Flash to USB

Write the ISO to a USB drive with dd or any compatible flashing tool.

bash
dd if=kernos-*.iso of=/dev/sdX bs=4M status=progress && sync
3

Boot & Install

Boot from the USB and run kerninstall command. Connect to WiFi with iwctl.

bash
kerninstall
1

Add the KernOS repo

Add the KernOS pacman repository to your /etc/pacman.conf.

/etc/pacman.conf
[kernos]
Server = https://github.com/palazik/linux-kernos/releases/latest/download
2

Install the kernel

bash
sudo pacman -Sy linux linux-headers
3

Update bootloader & reboot

bash
sudo grub-mkconfig -o /boot/grub/grub.cfg && reboot
1

Clone & install deps

bash
git clone https://github.com/palazik/KernOS && cd KernOS
sudo pacman -S archiso
2

Build the ISO

bash
sudo mkarchiso -v -w /tmp/kernos-work -o out/ profile/
3

Test in QEMU

bash
run_archiso -i out/kernos-*.iso
