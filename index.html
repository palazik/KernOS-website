// kernel

Built different, at the core

The KernOS kernel isn't just patched — it's rebuilt with every optimization that matters for desktop performance.

SCHEDULER BORE

Burst-Oriented Response Enhancer

Prioritizes latency-sensitive tasks without sacrificing throughput. Interactive applications feel snappier, compiles stay fast.

CONFIG_SCHED_BORE=y CONFIG_SCHED_CLASS_EXT=y
COMPILER ThinLTO + O3

Whole-Program Optimization

Built with Clang, Link Time Optimization across the full kernel, and -O3 for maximum instruction-level performance.

CC=clang LD=ld.lld LLVM=1 CONFIG_LTO_CLANG_THIN=y
TIMER 1000Hz + Tickless

Full Preemption, Zero Idle Ticks

1000Hz timer for microsecond responsiveness. NO_HZ_FULL eliminates unnecessary scheduler interrupts when cores are busy.

CONFIG_HZ_1000=y CONFIG_NO_HZ_FULL=y CONFIG_PREEMPT=y
NETWORK BBR + FQ

Google's Congestion Control

BBR TCP replaces CUBIC as default. Paired with FQ packet scheduler for significantly better throughput on all network types.

CONFIG_TCP_CONG_BBR=y CONFIG_DEFAULT_BBR=y
MEMORY Transparent HugePages

Always-On THP

THP set to always (not madvise). Reduces TLB pressure, improves memory bandwidth for large workloads and games.

CONFIG_TRANSPARENT_HUGEPAGE_ALWAYS=y
PATCHES CachyOS + Polly

Upstream Optimizations

Built on CachyOS's pre-patched kernel tree with ACPI call support and Clang Polly loop vectorization.

bore-cachy.patch clang-polly.patch acpi-call.patch

The custom kernel package

A separate AUR-compatible PKGBUILD that installs the KernOS kernel on any Arch-based system — no full reinstall needed.

📦

PKGBUILD based

Standard Arch Linux packaging. Install with makepkg -si or any AUR helper.

🔄

Auto-versioning

Automatically tracks the latest CachyOS kernel release. Always up to date.

🏗️

CI builds

GitHub Actions builds kernel packages on every commit. Download prebuilts from Releases.

🗄️

Pacman repo

A ready-to-use pacman repository is generated per build. Add to pacman.conf and install directly.

Install the kernel on any Arch system 
# Clone the kernel PKGBUILD
git clone https://github.com/palazik/linux-kernos
cd linux-kernos

# Build and install (requires: clang llvm lld base-devel)
makepkg -si --noconfirm

# Verify install
uname -r  # → 6.x.x-kernos
// getting started

Install KernOS

KernOS ships as a bootable ISO built on archiso. Boot, follow the Arch installation guide, and you're done.

01

Download the ISO

Grab the latest ISO from GitHub Releases. Built automatically by GitHub Actions on every commit to main.

GitHub Releases →
02

Write to USB

Flash the ISO to a USB drive. Supports both BIOS (syslinux) and UEFI (systemd-boot) boot modes.

dd if=kernos-*.iso of=/dev/sdX bs=4M status=progress
03

Boot the live environment

Boot into the live system. Networking starts automatically (Ethernet/WLAN/WWAN via DHCP). For Wi-Fi use iwctl.

04

Install with archinstall

Run the bundled archinstall script for a guided install, or follow the Arch installation guide manually. The MOTD shows the link on boot.

archinstall
05

Run in QEMU (optional)

Test the ISO without hardware using the built-in run_archiso helper.

run_archiso -i out/kernos-*.iso
// build it yourself

Reproducible ISO builds

Every ISO is built by GitHub Actions on an Arch Linux container. The full build is reproducible and open.

Build the ISO from source 
# Requirements
sudo pacman -S archiso

# Clone and build
git clone https://github.com/palazik/KernOS
cd KernOS
sudo mkarchiso -v \
  -w /tmp/kernos-work \
  -o out/ \
  profile/

# ISO lands in out/
ls out/kernos-*.iso

CI Pipeline

GitHub Actions runs on every push to main. The workflow:

  1. Spins up Arch Linux container
  2. Installs archiso and deps
  3. Generates ephemeral PGP signing key
  4. Builds the squashfs ISO
  5. Creates SHA256/SHA512/B2 checksums
  6. Uploads artifact (30-day retention)

ISO Profile

Based on archiso's releng profile. Boots via syslinux (BIOS) and systemd-boot (UEFI). Squashfs-compressed with xz.

archinstall fastfetch grml-zsh linux-firmware nvme-cli btrfs-progs +80 packages
